# Loan Approval Predictor

Binary classification ML project to predict loan approval.

## Models Used
- Logistic Regression → 87.5% accuracy ✅ (best)
- Naive Bayes → 86.5%
- KNN → 75.5%

## Tech Stack
Python, scikit-learn, pandas, numpy, matplotlib, seaborn

## Dataset
1000 samples, 20 features — income, credit score, DTI ratio, etc.
